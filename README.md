# Big Band Soc Amsterdam Tour - First Aid Info

A simple static website providing first aid and safety information for the Big Band Soc Amsterdam tour (June 1-5, 2026).

## Features

- Emergency contact information and protocols
- Hospital locations and details in Amsterdam
- First aid guidance for common issues
- Cannabis/greening out response procedures
- Committee sober rota with backup support
- Key contact information
- Mobile responsive design

## Usage

Open `index.html` in any web browser. No server or build process required.

Before the tour, fill in:
- Committee member names and phone numbers
- Sober rota assignments
- Jack's contact information
- Tour leader and accommodation details

## Important Notes

- **Jack** is the designated first aider and will not get too inebriated to help
- **Do not film** people who are drunk or experiencing medical issues
- Carry your **passport and GHIC card** at all times
- For serious emergencies: **Call 112 first, then contact Jack**
- For non-serious injuries: **Contact Jack immediately**
